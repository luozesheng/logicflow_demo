
### 功能
- [ ] AI后台 - 流程图的绘制插件
	- [ ] 支持流程图的交互事件
		- 点击事件，回传父级，进行弹窗，修改节点数据，以及
	- [ ] 支持html节点绘制
	- [ ] 支持vue组件的开发渲染方式
	- [ ] 支持条件窗口，能够下发给多级用户
	- [ ] 事件全局管理以及节点事件互传，父子级冒泡
	- [ ] 在处理过当前的画布节点后，重新生成data的数据集-导出数据集
	- [ ] 节点具备功能：
		- 锚点的个数需要动态设置
		- 作为html自定义节点，需要根据store中的节点数据进行修改其文本

1. 如何通过setAttributes对节点的属性进行设置
